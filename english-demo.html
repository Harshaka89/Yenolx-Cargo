Implementation Details

Translation Files

The English translations are stored in the /languages/en_US.po and /languages/en_US.mo files. These files contain all the translatable strings used throughout the plugin.

# English Translation Example msgid "Yenolx Cargo Service" msgstr "Yenolx Cargo Service" msgid "Select Country" msgstr "Select Country" msgid "Calculate Price" msgstr "Calculate Price"

Language Detection

The plugin automatically detects the current language based on URL parameters or user preferences. The default language is English.

// Language detection function function yenolx_get_current_language() { if (isset($_GET['lang'])) { return sanitize_text_field($_GET['lang']); } // Default to English return 'en'; }

Shortcodes

The plugin provides shortcodes that automatically include the language switcher and display content in the appropriate language.

Order Form: [yenolx_order_form]
Tracking Form: [yenolx_tracking_form]
Price Calculator: [yenolx_price_calculator]
